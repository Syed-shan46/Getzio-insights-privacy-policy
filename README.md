# Getzio Insights - Privacy Policy & Terms of Service Web Page

This is a premium static web project housing the App Store-compliant legal disclosures, data privacy statements, and account deletion protocols for **Getzio Insights**.

## Visual Design
Designed with a dark futuristic glassmorphism theme (`#09090b` charcoal grid, blurred neon orbs, white and grey typography) to match the mobile app's WWDC/Linear style guidelines.

## Quick Start Deployment

Deploy the site directly to **GitHub Pages** using the following steps:

1. **Initialize Git**:
   ```bash
   git init
   git add .
   git commit -m "Initialize Getzio Insights privacy policy web assets"
   ```

2. **Add Remote & Sync**:
   * Create a new public repository on GitHub named `Getzio-Insights-privacy-policy`.
   * Associate the remote and push:
   ```bash
   git remote add origin https://github.com/Syed-shan46/Getzio-Insights-privacy-policy.git
   git branch -M main
   git push -u origin main
   ```

3. **Enable GitHub Pages**:
   * Open the repository settings on GitHub.
   * Navigate to **Pages** in the sidebar.
   * Under **Build and deployment**, select **Deploy from a branch** and set it to `main` (`/root`).
   * Save. Your policy site will go live at `https://syed-shan46.github.io/Getzio-Insights-privacy-policy/` within a couple of minutes!
